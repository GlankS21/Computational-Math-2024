Структура программы.
1. Точка входа в программу находится в файле L2MainPoint. В ней создается экземпляр окна L2Frame (от JFrame).
2. В конструкторе L2Frame в этом окне размещаются и соединяются все компоненты интерфейса пользователя
    - Меню File с подменю:
        - Открыть файл с параметрами -> проецируется на метод L2Frame.openFile
        - Сохранить параметры в файле -> проецируется на метод L2Frame.saveParams
        - Сохранить результаты в файле -> проецируется на метод L2Frame.saveResults

    - Панель для отрисовки графиков - DrawPanel. Она размещается в центре и сохраняется под именем _dp. Для того, чтобы
    график был отрисован, необходимо вызвать _dp.setFunction() или _dp.setFunctions() и передать в качестве параметров
    IFunction[] тех графиков, которые должны быть отображены на панели.

    - панель параметров L2Params (от JPanel). При создании панели параметров L2Frame передает в L2Params методы,
    которые должны вызыаться панелью при нажатии на кнопки Рассчитать и Построить график.
    Методы для расчета L2Frame.Calculate и L2Frame.CalculateSys передаются с помощью интерфейса CalculationProcess
    в вызове метода L2Params.setCalcProcess.
    Для того, чтобы по нажатию кнопок "Нарисовать график" появлялся гарфик на экране, L2Frame передает L2Params через
    вызов метода L2Params.setPainter экземпляр интерфейса DrawGrafic, в котором определяются действия при вызове
    рисования графика функции или системы уравнений. В качестве действия указаны действия
        _dp.setFunction()
        _dp.repaint()
        для одного уравнения. И
        _dp.setFunctions()
        _dp.repaint()
        для системы уравнений. repaint() нужно вызвать, чтобы график сразу отобразился на панели.

        При инициализации панели с параметрами L2Params, у кнопок вызывается метод setActionListiner, в котором
        указывается, какой метод класса L2Params должен быть вызван при нажатии кнопки. Это методы
        - startCalculation - для расчета одного уравнения
        - startSysCalculation - для расчета системы уравнений
        - startDrawing - для отрисовки графика одного уравнения
        - startDrawGrafics - для отрисовки графиков системы уравнений
        Эти методы в свою очередь вызывают методы, ранее полученные от L2Frame -L2Frame.Calculate,L2Frame.CalculateSys и
        действия для отрисовки графиков, описанные выше.
